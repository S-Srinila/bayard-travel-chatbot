# Bayard Vacations Travel Assistant

A Streamlit-based chatbot that provides travel package details and booking guidance.

## Live Demo
https://bayard-travel-chatbot.streamlit.app

## Tech Stack
- Python
- Streamlit
- Rule-based NLP

## How it Works
- Intent detection from user input
- Package data lookup
- UI rendered with Streamlit and custom CSS
